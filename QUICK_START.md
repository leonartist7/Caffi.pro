# ☕ Caffi.Pro - Quick Start

## 🚀 Run Locally in 3 Steps

### 1. Install
```bash
npm install
```

### 2. Start
```bash
npm run dev:local
```

### 3. Open
```
http://localhost:3000
```

---

## 📍 Important URLs

- **Admin Dashboard**: http://localhost:3000/clients
- **Diagnostics Tool**: http://localhost:3000/diagnostics ← **Use this first!**
- **Sample Shop**: http://localhost:3000/shop/green-landscaping-services

---

## 🔧 Essential Commands

```bash
npm run dev:local     # Start development server
npm run build        # Build for production
npm run type-check   # Check TypeScript errors
npm run lint         # Check code quality
```

## ✅ Verify It's Working

Look for this on any page with a clock:
```
• Brew Excellence Daily ☕
```
✅ If you see this, you're running the latest version!

## 🐛 If Shop Pages Show "Not Found"

1. Go to Supabase SQL Editor: https://supabase.com/dashboard
2. Run the SQL script from `LOCAL_DEVELOPMENT_GUIDE.md` (Section: Database Setup)
3. Refresh your browser
