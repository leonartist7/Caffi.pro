# PWA Icons Required

For the PWA to work properly, you need to create the following icon files:

## Required Icons:

1. **icon-192.png** (192x192 pixels)
   - Square PNG image with coffee/cafe branding
   - Should be simple and recognizable at small sizes
   - Used for Android home screen

2. **icon-512.png** (512x512 pixels)
   - Same design as 192x192 but higher resolution
   - Used for splash screens and high-DPI displays

3. **favicon.ico** (16x16, 32x32, 48x48 multi-size)
   - Browser tab icon
   - Classic .ico format

## Temporary Solution:

You can use these free icon generators:

1. **Favicon.io** - https://favicon.io/
   - Generate from text, emoji (☕), or image
   - Downloads all required sizes

2. **RealFaviconGenerator** - https://realfavicongenerator.net/
   - Upload one image
   - Generates all PWA icons

## Quick Coffee Emoji Solution:

Use ☕ emoji on a brown background (#6b3410) to match your theme color.

## Where to Place:

All icon files go in the `/public` folder (same location as this file).
