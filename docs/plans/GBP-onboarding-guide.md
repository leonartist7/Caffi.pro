# Google Business Profile onboarding guide

A complete Google Business Profile helps nearby guests find the café and gives them a trusted path to the website, menu, and opening hours. Complete these steps after the aro website has been reviewed and published.

## 1. Claim or create the profile

1. Sign in at [Google Business Profile](https://www.google.com/business/) with an account the café owner controls.
2. Search for the exact café name and address before creating anything new.
3. Claim the existing listing when one exists. Create a new profile only when Google finds no match; duplicate listings split reviews and confuse search results.
4. Use the real public-facing business name without extra keywords or neighbourhood phrases unless they are part of the name used in the café.

## 2. Verify ownership

Google chooses the available verification method. It may offer video, phone, email, live video, or a mailed postcard. Follow the instructions shown in the profile and keep the verification code private. Do not change the business name or address while verification is pending unless Google specifically asks.

## 3. Match the aro website

- Add the published website URL shown in **aro → Settings → Website**. Use the custom domain when the café has one.
- Copy the same display address and phone number used on the aro website.
- Add the primary category that best describes the business, such as “Coffee shop” or “Cafe,” then add only accurate secondary categories.
- Use the same weekly hours configured in aro Reservations. Add holiday or exceptional hours directly in Google whenever the normal schedule changes.
- Add a short description in the same warm, plain-spoken voice as the website. Avoid promotions, prices, and links in the description.

## 4. Add useful photos

Upload a clear exterior photo, counter or interior photo, several current menu items, and the café logo. Use recent, well-lit images that accurately represent an ordinary visit. Replace outdated seasonal or menu photography instead of leaving it indefinitely.

## 5. Keep the listing healthy

- Review the profile monthly for suggested edits, incorrect hours, or duplicate listings.
- Update holiday hours before each closure or schedule change.
- Respond to new reviews at least weekly. Thank positive reviewers specifically; acknowledge problems without arguing and invite the guest to continue the conversation privately.
- Never offer rewards in exchange for positive reviews. Ask regulars for honest feedback using Google’s review link.
- Check that the website button still opens the published aro site after any domain change.

## Launch check

The listing is ready when the café is verified, its name/address/phone/hours match aro, the published website opens correctly, and at least one exterior, interior, product, and logo image is visible.
